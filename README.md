# Korp_Teste_OzyKleyton

Neste repositório há três aplicações:
- `estoque` (API Go)
- `faturamento` (API Go)
- `frontend` (Angular)

## Como executar

No diretório raiz do repositório execute:

```bash
./run.sh
```

Depois, abra no navegador:

```bash
http://localhost:4200
```

A partir daí, o frontend consumirá os backends configurados na mesma rede do Docker Compose.
